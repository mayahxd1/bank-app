# Four in a Row Game

## Description
A bank app software developed using JavaScript and React.

## Technologies Used
- JavaScript
- React
- Node.js
- MongoDB

## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/mayahxd1/bank-app.git
    cd bank-app
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the application:
    ```bash
    npm start
    ```
